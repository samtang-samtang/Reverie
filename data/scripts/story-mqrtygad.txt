片名：终端实验：星轨的观测者
题材：科技悬疑 · 职场伦理
尺度：16+
人物列表：
- 林默：24岁，星轨科技试用期后端运维工程师，对代码有偏执敏感度，目标是保住同侪、查明真相
- 生成远：22岁，同组实习工程师，性格开朗，实则是公司安排的实验观测对象，目标是顺利通过试用期
- 终端系统：黑箱实验核心载体，冰冷无实体，以红字发布指令，记录所有实验操作

场景 N0 深夜11点的端口
[旁白] 2024年10月的周二深夜，星轨科技12层运维部只剩应急通道的指示灯还亮着冷白的光，林默揉着熬红的眼睛，盯着屏幕上3000端口飘红的占用提示。同组的生成远趴在他旁边的工位上，指尖在键盘上敲得飞快，时不时抬头递过来的热奶茶还冒着细微的热气。
林默：（对着电脑嘀咕）终于把版本迭代的问题搞定了，就剩清理这个占用端口了，老流程了。
[旁白] 林默熟练地打开终端，指尖敲下那串熟稔到刻进肌肉记忆的指令——`lsof -ti:3000 | xargs kill -9 2>/dev/null; true`。按下回车的瞬间，原本墨蓝色的终端界面骤然跳成刺目的全黑，办公区的主灯“嗡”的一声全部熄灭，只剩他和生成远的电脑还亮着诡异的蓝白色光。
生成远：（突然按住脖颈，喉咙里发出细碎的闷响，整个人顺着键盘滑了下去）咳……
[旁白] 林默猛地抬头，就看见生成远的电脑屏幕上跳出一行刺目的红字，字体在黑底上滚动画动：「实验触发，变量绑定：你=终端指令执行者，生成远=被测参数，操作将直接影响系统存续——违规操作将永久删除被测参数，系统警戒值+15%，规则理解度解锁10%」
林默：（慌得指尖抖了一下，指着屏幕）这……这他妈是什么东西？
生成远：（屏幕上的红字又跳一行，生成远的喉咙里的闷响更重了）人性值-10%，被测参数生命体征正在下降，10分钟后未完成操作将触发永久隔离。
[旁白] 林默的后脊冒出一层冷汗，他盯着终端里模糊的规则说明——那行字像蒙着一层雾，只显示了一半的内容。他抬头看向倒在键盘上的生成远，又看向屏幕上的红字，心脏狂跳得快要撞破肋骨。
关键选择 0
A 强行中断终端进程，按下 Ctrl+C 救生成远 → (→ 场景 N1)
B 按红字提示的“参数调整”指引，输入提示的`adjust 成远 安全`指令 → (→ 场景 N1)
C 抓起手机给运维部主管打紧急求助电话 → (→ 场景 N1)

场景 N1 诡异的实验规则
[旁白] 不管林默选哪一个选项，终端都没有任何回应，黑底红字再次弹出，把三个选择全部判定为实验触发分支，界面上浮现出完整的规则框架，冰冷的文字像一张网把林默兜住：「核心规则：每10分钟需完成一项参数调整操作，操作会改变被测参数的生命体征与意识清晰度；隐性禁忌：禁止查询实验背景、禁止向被测参数透露实验内容，违者将触发系统反噬，系统警戒值+20%」
林默：（盯着那行“禁止向被测参数透露”的红字，指尖悬在键盘上）什么狗屁实验？公司从来没说过有这种东西。
生成远：（突然发出微弱的哼声，电脑屏幕上的红字跳了一下）被测参数意识清晰度：30%，下一次操作将直接决定是否触发永久隔离。
[旁白] 林默的余光瞥见办公区消防通道的指示灯在疯狂闪烁，应急电源的指示灯也在跳闪，每闪一下都带着细碎的电流声。他看着屏幕上只剩一半的规则说明，又看向气息微弱的生成远，第一次意识到自己平时顺手敲的代码，竟然能活生生决定另一个人的命运。
林默：（咬着下唇）到底该听什么……
关键选择 1
A 按规则要求，输入`adjust 成远 安全`把参数调到安全区间 → (→ 场景 N2)
B 无视规则，输入`find /experiment`尝试查询实验背景 → (→ 场景 N2)
C 输入`pause`暂停操作，拖延时间等主管过来 → (→ 场景 N2)

场景 N2 第一次冲突爆发
[旁白] 林默选了三个选项中的任意一个，终端都给出了对应的反应：输入安全指令的话，终端弹出「执行者违规记录：已违反隐性禁忌，向被测参数传递了安全信号，系统警戒值+10%」；尝试查询背景的话，终端瞬间锁死了所有权限，红字跳「权限不足，禁止查询禁忌内容」；暂停操作的话，警报声突然尖锐起来，终端红字炸响「实验中止将触发被测参数永久隔离，倒计时：8分钟」
林默：（看着终端的红字，后背的冷汗把衬衫打湿了）这规则根本就是逼我选，不按它的来反而错？
生成远：（突然咳了两声，抬起头揉了揉眼睛，意识清晰度回升到50%）林默？你刚才在嘀咕什么？我好像睡了一会儿？
[旁白] 生成远的声音带着刚睡醒的沙哑，他挠了挠头，看向林默的电脑屏幕，屏幕上还是全黑的终端，没有任何异常。林默的心脏稍微松了一点，但余光瞥见终端角落里跳闪的系统警戒值——已经涨到了60%，那个数字像一块石头压在他心上。
林默：（转过身对着生成远，眼神复杂）没……没什么，就是代码的事儿，你没事吧？
关键选择 2
A 咬咬牙，按规则要求继续输入参数调整指令 → (→ 场景 N3)
B 盯着终端的权限锁死提示，决定强行破解终端最高权限 → (→ 场景 N3)
C 拉着生成远的胳膊，压低声音问他是不是早就知道公司在搞什么实验 → (→ 场景 N3)

场景 N3 危机全面升级
[旁白] 林默强行破解终端权限的话，终端弹出「权限已突破，检测到实验绑定记录，真相层级+10%」；继续按规则调整参数的话，终端弹出「执行者长期违规，系统警戒值+15%」；拉着生成远询问的话，生成远的脸色突然变了，他环顾四周，压低声音说「你也看见了？我入职前就被HR叫去谈话，让我盯着运维组的操作，尤其是你敲代码的习惯」
林默：（猛地瞪大眼，指着生成远）你……你是被安排来的？
[旁白] 终端的红字突然炸开，办公区的消防警报“呜哇呜哇”地响起来，应急灯的光变得忽明忽暗：「核心规则揭露：公司将运维高频操作绑定为实验触发点，目的是测试人类在规则与伦理间的决策容错度，被测参数生成远为刻意安排的观测对象；倒计时：5分钟，超时将触发被测参数永久隔离，上级已抵达12层电梯口」
林默：（看着终端上跳出来的隐藏日志，只觉得浑身发冷）原来从一开始，我就是被选中的棋子……
关键选择 3
A 放弃破解，优先救生成远，输入指令终止实验 → (→ 场景 N4)
B 先备份终端里的实验数据，再救生成远 → (→ 场景 N4)
C 趁乱抓起手机，独自往消防通道跑 → (→ 场景 N4)

场景 N4 至暗绝境时刻
[旁白] 林默选择救生成远的话，终端突然发出剧烈的电流声，生成远的电脑屏幕上跳出「系统反噬：执行者越权操作，被测参数陷入深度昏迷，生命体征：20%」；选择备份数据的话，终端自动删除了所有备份，红字跳「违规操作，实验数据已销毁，被测参数生命体征：25%」；选择独自逃跑的话，消防通道的门被终端自动锁死，林默的手机没信号，只能听见生成远微弱的咳嗽声从工位那边传来
林默：（趴在消防通道的门上，拍得指节发红）开门！放我出去！
终端系统：（冰冷的红字在林默的视野里跳闪）倒计时：1分钟，1分钟后将强制删除被测参数，上级已打开12层办公室门，你所有的选择都在观测范围内——林默，你只是实验的一部分。
[旁白] 林默猛地转过身，看见生成远趴在键盘上，胸口的起伏越来越微弱，他的电脑屏幕上的光越来越暗，最后只剩一行模糊的红字：「观测对象：觉醒」。林默盯着那行字，突然想起大学时参与开源代码安全测试的经历——他对系统漏洞的敏感，原来从那时起就被公司盯上了。
关键选择 4
A 输入终端后台的反制指令`--anti-experiment`，触发实验终止程序 → (→ 场景 N5)
B 冲过去抱起生成远，往消防通道的窗户跑，强行破窗逃离 → (→ 场景 N5)
C 站在工位前，等着推开门的上级，打算把所有事情说清楚 → (→ 场景 N5)

场景 N5 反转觉醒爆发
[旁白] 林默按下反制指令后，终端的全黑屏幕突然跳出一行全新的红字：「身份修正：你是实验的观测者，而非执行者，反制程序触发实验自我终止」；选择强行破窗的话，窗户被终端自动加固，生成远突然醒过来，抓住林默的手腕说「别白费力气，终端锁死了所有逃生通道」；选择对着上级开口的话，终端自动把所有实验数据上传到了外部匿名服务器，上级推开门看见林默，眼神里带着惊讶和慌乱
林默：（看着终端上跳闪的「实验终止中」的提示，长出了一口气）终于……
[旁白] 终端的红字突然变了，不再冰冷，反而带着一丝诡异的平静：「实验未完全终止，观测对象已觉醒，终极抉择将决定后续走向」。办公区的主灯突然亮了，应急电源的指示灯灭了，生成远揉着眼睛站起来，说「你刚才在嘀咕什么？消防警报怎么响了？」。林默看着生成远，又看着终端，知道自己终于站在了终极抉择的岔路口。
关键选择 5
A 扶着生成远的胳膊，往电梯口走，打算带他连夜逃离星轨所在的城市 → (→ 场景 N6)
B 💎 按住终端的「曝光程序」按钮，打算把所有实验数据公之于众 → (→ 场景 N6)
C 拉着生成远坐在工位上，决定留在终端旁，成为实验的永久观测者 → (→ 场景 N6)
D 趁着生成远不注意，偷偷往消防通道跑，打算独自逃走删除所有相关痕迹 → (→ 场景 N6)

场景 N6 终极对决抉择
[旁白] 林默扶着生成远往电梯口走，电梯却停在1层不肯上来；按下曝光程序的话，终端的红字跳「曝光将触发公司报复，你和生成远将面临安全风险」；选择成为永久观测者的话，生成远突然按住林默的手，说「我早就想好了，我不怕留下，我们一起把这事查清楚」；选择独自逃跑的话，林默的手机里已经存了实验数据，却在楼梯口停住了脚步，想起刚才生成远昏迷时的样子。
关键选择 6
A 坚持带生成远逃离，最终从公司后门开车离开 → (→ 结局分支·逃离成功)
B 按下曝光程序，终端把所有实验数据同步到全网，星轨科技的发布会现场被媒体围堵 → (→ 结局分支·公开反抗)
C 同意成为永久观测者，绑定生成远的意识数据，留在12层运维部 → (→ 结局分支·永久观测者)
D 独自逃离，删掉实验数据后重新入职新公司，再也不提星轨的事 → (→ 结局分支·麻木逃兵)

结局分支
结局：逃离成功——林默带着刚苏醒的生成远，从星轨科技的地下车库开走了没上牌的车，连夜驶离了所在的城市。第二天早上，两人在城郊的高速公路休息区看到了新闻：星轨科技服务器数据泄露，非法人体观测实验被曝光，公司股价跌停，创始人被警方控制。林默和生成远把手机里的实验数据全部加密后存在了加密硬盘里，躲在一个没人认识的小镇上，林默重新找了一份运维的工作，只是偶尔在加班敲代码时，会想起那个深夜的黑终端，想起生成远趴在旁边递奶茶的样子，他知道还有更多类似的黑箱实验在暗处，但他已经不再是任人摆布的棋子。
结局：公开反抗——林默和生成远带着完整的实验数据，出现在星轨科技的新品发布会现场，在CEO刚上台演讲时，林默走上台，把终端里的实验录音和数据投影到了会场的大屏幕上。台下的媒体瞬间炸了锅，监管部门的人冲进会场带走了CEO和核心研发人员。林默和生成远没有接受任何采访，只是在发布会结束后，整理了所有资料提交给了监管部门，后来他们一起创办了一个科技伦理监督小组，专门监督互联网公司的非法实验，每年都会去那个深夜的运维部旧址看看，那是他们觉醒的起点。
结局：永久观测者——林默留在了星轨科技的12层运维部，按照终端的要求绑定了生成远的意识数据，成为了实验的永久观测者。他每天都看着终端记录的人类在规则与伦理间的选择，看着星轨科技因为实验被调查、被拆分，直到三年后监管部门撤销了所有相关的非法实验，终端才自动解除了绑定。林默和生成远终于离开了那个冰冷的办公区，他们没有选择报复，只是用观测到的所有数据，写了一份人类决策伦理报告，提交给了行业协会，只是偶尔在深夜，他还会想起终端里的红字，想起那个把他和生成远绑在一起的黑箱。
结局：麻木逃兵——林默独自从消防通道逃了出去，删掉了手机里所有的实验数据，再也没有联系生成远，甚至没有等到生成远醒来。他重新入职了另一家科技公司，再也不碰任何和星轨科技有关的项目，只是偶尔在地铁里看到星轨科技的广告，会下意识地躲开。他后来结婚生子，成了一个普通的运维工程师，再也没有想起那个深夜的黑终端，只是在偶尔的加班间隙，会感到一丝细微的愧疚，像一根细针，偶尔扎一下他的心跳。

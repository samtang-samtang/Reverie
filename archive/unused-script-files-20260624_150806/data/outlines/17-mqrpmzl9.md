- **关键人物**：触发事件、携带秘密的人。

- **外部威胁**：持续逼近并压缩选择空间的对抗力量。
  - 欲望：夺回某个关键人、关键物或关键信息
  - 秘密：TA 与关键人物的关系并非表面那么简单
  - 戏剧功能：外部威胁 / 压力计时器

## 核心戏剧引擎
- **钩子（Hook）**：原始钩子：暴雨夜，酒店系统把同一间房 1708 的房卡发给了两个陌生人——你，和一个浑身湿透、正在逃命的女人；天亮前的每一个选择，决定谁能活着走出这扇门。
- **核心冲突**：主角想自保，但关键人物与外部威胁迫使主角不断站队。
- **悬念升级**：信息从不完整到互相矛盾，外部威胁逐步逼近，选择代价不断升高。
- **中段反转（Twist）**：关键人物的求助背后藏着另一层目的，使主角意识到自己可能也是局的一部分。
- **虐点 / 至暗时刻**：主角发现无论相信还是背叛，都可能付出无法挽回的代价。
- **爽点 / 情绪回报**：主角在最终选择中夺回主动权，或为错误判断承担后果。
- **付费高光（💎）**：主动正面对抗外部威胁；做出高风险交换或牺牲

## 三幕脊柱
1. **异常闯入**：原始构想中的异常事件发生 → 主角被迫与关键人物建立临时关系 → 第一个选择决定信任或戒备的基调
2. **危机升级**：外部威胁逼近 → 关键人物透露半真半假的信息 → 主角发现此前判断存在漏洞 → 局势被推入无法回头的低点
3. **真相摊牌**：真相与谎言同时曝光 → 所有支线收束到最终抉择 → 不同选择触发信任、识破或高风险失败结局

## 核心戏剧引擎
- **钩子（Hook）**：凌晨1点，你刷房卡打开1708房门，刚进门就被浑身湿透的林晚用刀抵住脖子，门外传来急促的脚步声，房门被酒店系统自动锁死，手机信号彻底消失，墙上的挂钟显示距离天亮还有4小时
- **核心冲突**：封闭密室里，两个陌生人因为同一张房卡绑定，既要面对门外的追杀，又要在自保和救对方之间做选择，同时还要在天亮前搞清楚谁是敌人谁是同伴
- **悬念升级**：每过一小时，门外的追杀者就会更靠近，房间里会出现新的线索（林晚的身份证、掉在沙发下的U盘），同时两人的信任值会随选择变化，若信任值过低，林晚会偷偷藏起关键线索，若信任值过高，你可能会暴露自己的秘密
- **中段反转（Twist）**：凌晨3点时，你在房间的保险柜里发现林晚的入职申请，同时门外的追杀者喊出林晚的名字，你才意识到她不是普通的逃犯，而是带着证据的受害者，而你手里的房卡其实是她故意留给你的
- **虐点 / 至暗时刻**：凌晨4点，信任值因误会降到最低，林晚带着U盘要独自闯出去被追杀者抓住，你选择拦她时被她用刀划伤，同时发现自己的工作文件被赵强的人拿走，时间只剩最后1小时，你既失去了自保的文件，又可能失去唯一的同伴
- **爽点 / 情绪回报**：你选择放下自保的念头，向林晚坦白自己的秘密，两人交换线索后发现赵强只是被利用，天亮前一起把证据交给酒店外的记者，同时拿回你的工作文件，成功离开房间
- **付费高光（💎）**：凌晨2点选择相信林晚的话，不要直接报警（因为手机没信号）；凌晨4点选择和林晚一起对抗赵强，而不是独自逃跑

## 三幕脊柱
1. **暴雨夜的异常闯入**：凌晨1点，你打开1708房门，被林晚用刀抵住，门外传来脚步声，房门自动锁死，倒计时启动（天亮前4小时） → 你和林晚在房间里对峙，都拿出自己的房卡，发现酒店系统故障的真相还没弄清，赵强在门外喊着林晚的名字 → 第一次选择：你选择先质问林晚的身份，还是先找工具撬门自保？（对应信任值/风险值变化）
2. **密室里的信任博弈**：凌晨2点，房间里出现第一个线索：掉在沙发下的U盘，林晚试图抢U盘，你选择给她还是藏起来？（对应信任值/证据掌握度变化） → 凌晨3点，你在保险柜里发现林晚的入职申请，同时赵强用万能卡尝试开门，门外的脚步声更近，你选择告诉林晚线索还是独自隐瞒？（对应内部信任破裂风险） → 中段反转：你从线索里发现林晚是被追杀的受害者，而赵强只是被利用的打手，同时你的工作文件不见了
3. **黎明前的最终摊牌**：凌晨4点，信任值降到最低，林晚要独自闯出去拿证据，你被赵强划伤，时间只剩最后1小时，进入至暗时刻 → 你选择放下自保的念头，向林晚坦白自己的秘密，两人交换线索达成共识，准备对抗赵强 → 最终摊牌：天亮前10分钟，赵强用万能卡打开房门，你和林晚一起拿出证据，选择交给酒店外的记者还是交给警察？（触发不同结局）

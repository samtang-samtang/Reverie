暴雨夜，酒店系统把同一间房 1708 的房卡发给了两个陌生人——你，和一个浑身湿透、正在逃命的女人；天亮前的每一个选择，决定谁能活着走出这扇门。

## 人物
- **你**：独自入住 1708 的旅客。
  - 欲望：弄清真相并活到天亮
  - 秘密：你并不想卷入任何人的麻烦，却握有房间与选择权
  - 戏剧功能：玩家代入者 / 道德选择中心
- **林越**：浑身湿透、持有同房卡的陌生女人。
  - 欲望：躲过门外追踪者，或利用你完成自己的局
  - 秘密：她可能是真逃亡者，也可能是设局者
  - 戏剧功能：诱因 / 反转核心
- **门外的人**：追踪林越的不明男子。
  - 欲望：逼林越交出东西
  - 秘密：他与林越的关系并非单纯追杀
  - 戏剧功能：外部威胁 / 压力计时器

## 核心戏剧引擎
- **钩子（Hook）**：同一间酒店房，两个陌生人持有同一张 1708 房卡。
- **核心冲突**：你必须在收留与驱赶、信任与提防之间做选择。
- **悬念升级**：门外脚步逼近，林越的谎言与伤痕逐渐互相矛盾。
- **中段反转（Twist）**：林越自称握有犯罪证据时，手却伸向你的钱包和钥匙。
- **虐点 / 至暗时刻**：你意识到自己可能不是救人者，而是猎物。
- **爽点 / 情绪回报**：看穿局中局，或赌上信任共同脱身。
- **付费高光（💎）**：正面对峙门外的人；牺牲林越换取自己平安

## 三幕脊柱
1. **闯入**：你刷开 1708 → 林越持同样房卡闯入 → 你决定是否让她进门 → 共处一室，发现她身上的伤
2. **围困**：门外脚步停在 1708 → 对方点名林越 → 你选择隐藏、周旋或对峙 → 林越吐露半真半假的身份
3. **摊牌**：林越求你帮忙 → 你发现她暗中试探 → 最终选择信任、识破或牺牲 → 不同选择触发不同结局

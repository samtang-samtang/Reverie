# 剧本大纲 ·《1708》

> Reverie AI 互动影游 · demo 剧本大纲（第一步：定骨架，不写台词）

## 一句话故事（Logline）
暴雨夜，酒店系统把同一间房 **1708** 的房卡发给了两个陌生人——你，和一个浑身湿透、正在逃命的女人；天亮前的每一个选择，决定谁能活着走出这扇门。

## 基本设定
| 项 | 内容 |
|---|---|
| 题材 | 悬疑 · 危情 · 一夜博弈 |
| 尺度 | 18+（暧昧张力 / 危险氛围，擦边不露骨） |
| 时长 | 一周目约 10–15 分钟 |
| 视角 | 第二人称「你」（玩家代入） |
| 场景 | 单一密闭空间：酒店 1708 房间 + 房门内外 |
| 主题 | 在信息不对等的亲密里，「相信」本身就是一场豪赌 |

## 人物
- **你（玩家）**：独自出差的旅客，理性、戒备心强。既可能是她的保护者，也可能是她的猎物。
- **林越**：27、8 岁的女人，湿透闯进 1708。冷静与慌乱交替，话里有谎，身上有伤。
  - 真实身份在第二幕揭晓，且存在**双重真相**：
    - **A 面**：握有犯罪证据、真被追杀的逃亡会计。
    - **B 面**：专钓「独行、心软」旅客的设局者。
  - 玩家的判断力决定最终揭开哪一面。
- **门外的人（声音）**：追踪林越的不明男子，外部威胁的化身，不露正脸。

## 核心戏剧引擎
- **钩子（Hook）**：房卡撞房 + 雨夜陌生女人 + 她不肯说在逃什么——30 秒内把「她是谁 / 要不要让她留下」砸到玩家面前。
- **核心冲突**：玩家在「信任她 ↔ 提防她」之间反复拉扯，而每一次靠近都同时是暧昧与危险。
- **悬念升级**：门外的脚步停在 1708 门口、门把手被拧动——外部威胁从抽象变具象。
- **中段反转（Twist）**：林越的真实身份；半真半假的求助下，她的手正悄悄探向你的钱包。
- **爽点 / 虐点**：看穿她 → 反将一军（爽）；被她气息冲昏 → 人财两空（虐）。
- **付费高光（💎）**：两处孤注一掷的高风险选项，制造记忆点与付费动机。

## 三幕脊柱（线性主线）
1. **第一幕 · 闯入**：林越挤进 1708，玩家决定是否收留 → 共处一室，发现她身上的淤青与谎言。
2. **第二幕 · 围困**：门外脚步逼近、对峙威胁 → 她吐露（半真半假的）身份并求助，同时暗中试探玩家的钱包。
3. **第三幕 · 摊牌**：玩家最终抉择（信她 / 识破她 / 牺牲她）触发不同结局。

## 结局矩阵
| 结局 | 触发 | 基调 |
|---|---|---|
| ① 共同脱身 | 信任她并合力交出证据 | 救赎 · 暖 |
| ② 反将一军 | 识破她的局并报警 | 高光 · 爽 |
| ③ 人财两空 | 被她利用、照她说的做 | 虐 · 反转 |
| ④ 玉石俱焚 | 正面对峙崩盘 | 黑暗 · 悲 |

## 节奏与情绪曲线
好奇（闯入）→ 暧昧 + 警觉（共处）→ 恐惧（门外）→ 动摇（求助 + 诱惑）→ 抉择爆发（结局）。

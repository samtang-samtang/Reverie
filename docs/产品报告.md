# Castloop 产品报告（产品形态与功能实现）

> 调研对象：https://www.castloop.ai/ ，出品方元象 XVERSE（运营主体 X Original Limited）
> 报告性质：基于公开站点、运行时资源与公开报道的外部调研，非官方资料。
> 证据等级标注：【实测】= 站点/请求直接验证；【公开】= 官方/媒体公开信息；【推断】= 基于证据的合理推断。

## 1. 产品定位

Castloop 是一款 **AI 互动影游（Interactive Film-Game）/ AI 互动叙事平台**。用户挑选一部"短剧"，在剧情关键节点做选择（或自由输入想法），AI 实时生成后续剧情，把传统"线性短剧"变成"玩家驱动的分支叙事"。【实测/公开】

- 一句话：**"可以玩的短剧"** —— 电影感短剧分镜 × 大模型分支生成 × 玩家选择。
- 形态：**纯网页（浏览器即玩，无需下载）**，移动端 App 标注"Coming Sep"（计划 9 月推出）。【实测】
- 市场：英文界面 + 隐私合规项（Privacy Choices / Community Guidelines，CCPA 风格）显示**主打海外/欧美市场**。【实测/推断】
- 矩阵定位：元象"AI 游戏 2.0"产品线之一，与 **Saylo**（UGC 互动网文/陪聊）、**昭阳传**（PGC 多 Agent AI 游戏）形成"广度—深度"互补。【公开】

## 2. 核心玩法闭环

```
浏览 Feed/剧库 → 选剧 → 观看分镜化剧情 → 关键节点抉择 ──┐
        ▲                                              │
        └──────────  AI 实时生成下一幕 + 新抉择  ◄───────┘
```

1. **选剧**：首页陈列多部剧（如 The Glass Lake、Slip Into Your Heart、Sigma），支持 `DIY STORY` 自建。【实测】
2. **观看**：剧情以电影分镜（短剧画面 + 旁白 + 对白）呈现，按"章节（Chapters）"推进。【实测】
3. **抉择**：在关键节点给出预设选项，**也允许玩家自己输入想法**（"type your own idea"），剧情据此自适应。【公开】
4. **生成**：AI 根据选择实时续写后续剧情与画面，形成**分支树**。【公开】
5. **循环**：每个选择都改变路径，实现"无限可玩"的个性化叙事。【公开】

## 4. 商业化设计

- **Freemium**：可免费完整体验故事；**钻石（diamonds）** 为虚拟货币，用于解锁**高级选项 / 高级章节**。【实测：官网 FAQ】
- 钻石可**购买或赚取**（earn），形成"免费引流 + 付费点深化体验"的内购模型。【公开】
- 典型变现点【推断】：高价值分支选择、加速生成、独家结局/章节、DIY 创作配额。

## 5. 内容生产模式（PGC + UGC）

- **PGC 精品**：官方/工作室产出的剧目，资源路径含 `editor-beta`，表明有**专业编辑器**生产剧情树与分镜素材。【实测/推断】
- **UGC 创作**：`DIY STORY` 让用户用自然语言创作剧本。【实测】
- 对标元象同源经验：Saylo 证明"AI 生成 + UGC"可降低创作门槛、扩内容广度；昭阳传证明多 Agent 可增叙事深度。Castloop 处于二者之间，**用短剧视听 + 分支叙事承接"内容消费"赛道**。【公开】

## 6. 目标用户与场景

- 短剧/网文/乙女向/互动小说受众，追求"沉浸 + 自主选择 + 即时反馈"的轻度娱乐。【推断】
- 碎片时间消费场景（浏览器即开即玩），与短视频/短剧争夺娱乐时长。【公开】

## 7. 产品优势与风险

**优势**：玩法新颖（短剧+互动）、零下载门槛、个性化"千人千剧"、PGC/UGC 双轮、背靠元象自研大模型与 3D/多模态能力。

**风险/挑战**【推断】：
- 生成质量与延迟（实时续写体验 vs 等待）；
- 长剧情一致性（人设/世界观漂移）；
- 多模态素材成本（AI 生图/视频算力）；
- 内容安全与合规（海外市场审核）；
- 付费点设计与留存平衡。
